# Basic Particles

While they are used individually, you will also see many of the right hand particles here used as patterns in more complicated briefs moving forward.

- `ㅇㅗㅏ`: {^}와 / {^}과
- `-ㄹ`: {^}을 / {^}를
- `-ㅇㄴ`: {^}은
- `-ㄴ`: {^}는
- `-ㅇ`: {^}의
- `ㅇㅓㅣ`: {^}에
- `ㅇㅣ`: {^}이
- `ㅇㅜㅓ`: {^}워
- `-ㅎ`: {^}히
- `-ㅁ`: {^}며
- `-ㄷ`: {^}도
- `-ㅅ`: {^}서
- `-ㅈ`: {^}지
- `-ㄱ`: {^}고
